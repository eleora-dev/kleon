Privacy Policy

Eleòra Kleon is a system maintenance utility for Fedora Linux, optimized for KDE Plasma. It updates packages, removes obsolete kernels, and cleans logs, cache, temporary files and browser data.

This application does not collect, store, transmit, or share any personal data. All operations are performed locally on your machine. No data ever leaves your device.

Eleòra Kleon requires elevated privileges (via pkexec) solely to perform the following system-level operations:

  • DNF / Flatpak Update packages, remove unused ones, and clean package caches (including libdnf5 and system-upgrade data).
  • Kernel removal Remove obsolete kernel versions beyond the latest two.
  • systemd journal Reduce journal size to a configured limit.
  • Log files Delete rotated and compressed logs in /var/log.
  • Core dumps Remove crash dump files in /var/lib/systemd/coredump.
  • Crash reports (system) Remove ABRT system crash reports in /var/spool/abrt and /var/tmp/abrt.
  • PackageKit cache Clear the PackageKit package cache.
  • Temporary files Remove stale files from /tmp (older than 1 day) and /var/tmp (older than 7 days).
  • Bash history (root) Clear the root shell history file.

User-level operations run without elevated privileges and only affect the current user's home directory:

  • User cache Clean ~/.cache (KDE essentials are preserved) and __pycache__ folders in the home directory.
  • Bash history (user) Clear the user shell history file.
  • Recent documents Remove KDE recent document entries and the GTK recently-used.xbel file.
  • Crash reports (user) Remove ABRT user crash reports in ~/.local/share/abrt.
  • Browser data Clear cache, history and sessions for Brave, Chrome and Firefox. Saved passwords are only removed if the user explicitly enables the dedicated option.

No analytics, no tracking, no network requests. The source code is publicly available on GitHub.

Informativa sulla Privacy

Eleòra Kleon è un'utility di manutenzione del sistema per Fedora Linux, ottimizzata per KDE Plasma. Aggiorna i pacchetti, rimuove kernel obsoleti e pulisce log, cache, file temporanei e dati browser.

Questa applicazione non raccoglie, salva, trasmette o condivide alcun dato personale. Tutte le operazioni vengono eseguite localmente sul dispositivo. Nessun dato lascia il computer dell'utente.

Eleòra Kleon richiede privilegi elevati (tramite pkexec) esclusivamente per eseguire le seguenti operazioni di sistema:

  • DNF / Flatpak Aggiornare i pacchetti, rimuovere quelli inutilizzati e pulire le cache (incluse libdnf5 e i dati di system-upgrade).
  • Rimozione kernel Rimuovere le versioni kernel obsolete oltre le ultime due.
  • Journal systemd Ridurre la dimensione del journal entro un limite configurato.
  • File di log Eliminare i log ruotati e compressi in /var/log.
  • Core dump Rimuovere i file di crash in /var/lib/systemd/coredump.
  • Crash report (sistema) Rimuovere i report di crash ABRT di sistema in /var/spool/abrt e /var/tmp/abrt.
  • Cache PackageKit Svuotare la cache dei pacchetti di PackageKit.
  • File temporanei Rimuovere i file obsoleti da /tmp (più vecchi di 1 giorno) e /var/tmp (più vecchi di 7 giorni).
  • Cronologia Bash (root) Cancellare il file di cronologia della shell root.

Le operazioni utente vengono eseguite senza privilegi elevati e riguardano esclusivamente la home directory dell'utente corrente:

  • Cache utente Pulire ~/.cache (i file essenziali di KDE vengono preservati) e le cartelle __pycache__ nella home.
  • Cronologia Bash (utente) Cancellare il file di cronologia della shell utente.
  • Documenti recenti Rimuovere i documenti recenti di KDE e il file recently-used.xbel di GTK.
  • Crash report (utente) Rimuovere i report di crash ABRT utente in ~/.local/share/abrt.
  • Dati browser Pulire cache, cronologia e sessioni di Brave, Chrome e Firefox. Le password salvate vengono rimosse solo se l'utente abilita esplicitamente l'opzione dedicata.

Nessuna analisi, nessun tracciamento, nessuna richiesta di rete. Il codice sorgente è pubblicamente disponibile su GitHub.
